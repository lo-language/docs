# Lo

Lo is a *message-based* programming language designed for building robust concurrent applications in the real world â€“ where failures happen and security matters.

Lo is implemented as a transpiler to JavaScript and can thus target the browser, server, or desktop.

## Getting Started

1. Clone the source repo
2. Link the package globally
	
	```
	cd lo
	sudo npm link
	```
3. Test the installation

	```
	cd examples
	lo factorial.lo
	```
